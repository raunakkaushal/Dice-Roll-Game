Game Rules

By pressing the "Click Me" button
you'll roll both dice.

If Player 1 gets a bigger value on dice
🎲 "Player 1 is the Win" 🎲

If Player 2 gets a bigger value on dice
🎲 "Player 2 is the Win" 🎲

If both players get the same value
🎲 It's a tie! 🎲

The game ends when either player wins 5 times.
The player who reaches 5 wins first
🎲 That is the overall winner! 🎲
